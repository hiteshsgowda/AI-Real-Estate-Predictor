# AI Real Estate Price Predictor

A machine learning model built using Python and Scikit-learn to predict property prices based on features like square footage, bedroom count, and the age of the property.

## Tech Stack & Concepts Used
- Language: Python
- Libraries: Pandas, Scikit-learn
- Algorithm: Linear Regression
- Concepts: Supervised Learning, Train-Test Split, Predictive Modeling

## Results
The model successfully splits training and testing data to predict real estate valuation with high accuracy, generating custom evaluations for unseen property dimensions (e.g., predicting a price of $354.0k for a 2100 SqFt, 3 BHK, 4-year-old home).
